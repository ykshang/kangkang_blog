# 对象的遍历

``` js
const obj = {
  name: 'Alice',
  age: 25,
  gender: 'female'
};

```

## 1. for...in

``` js
for(let key in obj) {
  console.log(key + ': ' + obj[key]);
}
// name: Alice
// age: 25
// gender: female
```
## 2. Object.keys() 

``` js
const keys = Object.keys(obj);
keys.forEach(key => {
  console.log(key + ': ' + obj[key]);
});

// name: Alice
// age: 25
// gender: female
```

## 3. Object.getOwnPropertyNames() 

``` js
const keys = Object.getOwnPropertyNames(obj);
keys.forEach(key => {
  console.log(key + ': ' + obj[key]);
});

// name: Alice
// age: 25
// gender: female
```
