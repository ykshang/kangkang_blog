# 香菇蒸滑鸡

## 材料准备

| 用料   | 数量       | 用料    | 数量      |
|:---- |:-------- |:----- |:------- |
| 鸡全腿  | 1        | 盐     | 一勺      |
| 干香菇  | 5-10个均可  | 糖     | 一勺      |
| 生抽   | 2-3勺     | 耗油    | 半勺      |
| 葱段   | 2-3个     | 姜丝    | 少量      |
| 老抽   | 半勺       | 香菜    | 一大棵     |
| 地瓜淀粉 | 3勺/或者两大勺 | 料酒/白酒 | 两勺或白酒一勺 |
| 鸡蛋   | 一个       | 胡椒粉   | 2勺      |
| 食用油  | 四大勺约50ml |       |         |

## 食材处理

- 干香菇清洗干净后，温水泡发，泡十分钟后上锅小火煮三分钟。目的是跑得快，可以有效祛除脏东西和病菌等，然后放在一遍沉淀冷却。

- 鸡腿去骨，剁大丁，放入盆中，加水浸泡，适当加一些盐，泡出血水，洗净控干水分。

- 准备葱段，姜丝少许

## 制作步骤

- 将控干水分的鸡肉放到大碗里，加入两勺胡椒粉、一勺盐、半勺耗油、两勺料酒后抓匀

- 在鸡肉中加入两大勺地瓜粉，一个鸡蛋抓匀

- 将泡发好的香菇切小块，洗净加入碗中，抓匀，加入50ml食用油，拌匀，腌制十分钟左右。

- 坐锅，等水开后，在碗里加入100ml左右香菇水，撒上姜丝葱段，上锅蒸20-30分钟。高压锅10分钟即可

- 出锅按照个人口味撒上香菜即可

## 要点总结

- 白酒比料酒更合适

- 多蒸一会，把肉蒸紧实一点比较好吃

- 干香菇比鲜香菇味道更浓郁

- 干香菇煮一会，可以更有效的杀死可能存在的细菌寄生虫，还有残留的杂质
