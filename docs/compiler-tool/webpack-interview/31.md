# 31、如何处理图片？

### 图片文件

- 安装 file-loader 或 url-loader：`npm install file-loader` 或 `npm install url-loader`
- 在 Webpack 配置文件中配置加载器：

```javascript
module: {
  rules: [
    {
      test: /\.(png|jpg|gif)$/,
      use: [
        {
          loader: "file-loader",
          options: {
            name: "[name].[hash].[ext]", // 文件指纹配置
            outputPath: "images/", // 输出目录
          },
        },
      ],
    },
  ];
}
```

以上配置将处理以 .png、.jpg 或 .gif 结尾的图片文件，并将它们输出到指定的目录中。
