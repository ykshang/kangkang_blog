# git log

::: tip 主要作用
显示提交日志
:::

## 语法

```bash
git log [<options>] [<revision range>] [[--] <path>…​]
```

## 常用的参数

| 参数      | 解释              |
|:------- |:--------------- |
| --graph | 以Ascii的形式显示日志记录 |

## 常用命令举例：

- 输出当前日志
  
  ```bash
  git log
  ```

- 以Ascii的形式显示日志记录
  
  ```bash
  git log --graph
  ```
